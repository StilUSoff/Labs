Написать собственный класс, который содержит коллекцию из виджетов (фиксированного, либо динамического размера).
Создать метод динамического добавления элементов интерфейса в коллекцию, а также метод для соединения между собой помещенных в коллекцию виджетов с помощью механизма сигналов/слотов, чтобы действие с одним из них отображалось на других. Через функции отладки проверить, что соединения между объектами не дублируется многократно, и отсутствуют "висячие" соединения
Использовать виджеты следующих классов для соединения: QLabel, QSlider, QScrollbar, QSpinBox, для определения к какому классу относится объект использовать метаинформацию (см. Лекцию 7).
